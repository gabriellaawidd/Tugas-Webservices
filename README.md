# Film RESTful API
Simple RESTful API built with Node.js and Express.js for managing film data.
This project is created for testing RESTful API using in-memory data (without database)

## How to Run the App
1. Install dependencies (npm install)
2. Run the server (node app.js)
3. API will run at http://localhost:3000

## Notes: 
- No database is used
- Data will reset when the server restarts
